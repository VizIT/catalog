Visualization Catalog

Think of it as clip art for visualizations.

Click any of the catalog entries to see a live example with its HTML markup. The markup loads the visualization from the GitHub repository and can easily be incorporated into any web content. Each visualization is contained within a small web page for easy inclusion as an iframe in larger content making these visualizations a drop-in replacment for images.

Each visualization is accessed through a URL like 

     http://vizit.github.com/catalog/name/index.html

By default the visualization is displayed in a 200px x 200px canvas. A different size is specified by appending URL parameters to the request. This requests a 400px x 400px visualization of a dipole. 

     http://vizit.github.com/catalog/dipole/index.html?canvasHeight=400&canvasWidth=400
